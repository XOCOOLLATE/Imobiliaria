# Site CCS Gesso

Primeira versao do site da loja, feita com HTML, CSS e JavaScript puro.

## Como abrir no VS Code

1. Abra o VS Code.
2. Clique em `File` > `Open Folder`.
3. Escolha esta pasta do projeto.
4. Abra o arquivo `index.html`.
5. Clique com o botao direito no arquivo e escolha `Open with Live Server`.

## Onde alterar cada coisa

- `index.html`: textos, secoes, menus e estrutura da pagina.
- `css/style.css`: cores, tamanhos, espacamentos e aparencia.
- `js/script.js`: lista de produtos, busca e links de WhatsApp.
- `img/`: imagens do site.

## Ajuste importante

No arquivo `js/script.js`, troque este valor pelo numero real da loja:

```js
const numeroWhatsApp = "5500000000000";
```

Use o formato com codigo do pais e DDD, sem espacos, parenteses ou tracos.
